# 大学個別ファイルの仕様

## calendar

| フィールド | 型 | 説明 |
| --- | --- | --- |
| `name` | string | カレンダー表示名。 |
| `universityCode` | string | 紐づく大学コード。`/universities/{id}.code` と一致。 |
| `hasSaturdayClasses` | boolean | 土曜授業の有無。 |
| `inputUrl` | string | 入力データ。通常公式ホームページのURLになる。 |
| `date` | string(yyyy-mm-dd) | 入力データが作成された日付 |


# terms
| フィールド | 型 | 説明 |
| --- | --- | --- |
| `name` | string | 学期名称。必須。 |
| `{name}.shortName` | string | 学期の短縮名。 |
| `{name}.classCount` | number | 学期内の授業回数など集計値。 |
| `{name}.isHoliday` | boolean | 春休み、夏休み、冬休みの場合true |

ex.
yaml形式で記述する。

calendar:
 name: 学事予定
 hasSaturdayClasses: true
 inputUrl: https://waseda.co.jp/calendar/2026
 date: 2026-01-15
terms:
 - name: 第1クォーター
   shortName: 1Q
   classCount: 7
   isHoliday: false
 - name: 第2クォーター
   shortName: 2Q
   classCount: 7
   isHoliday: false
 - name: 春休み
   isHoliday: true
 - name: 夏休み
   isHoliday: true
 - name: 冬休み
   isHoliday: true
